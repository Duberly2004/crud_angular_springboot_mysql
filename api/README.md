# Student-Course API
![Vista previa de la API](public/preview.png)

Esta es una API para gestionar estudiantes y cursos, así como las relaciones entre ellos.

- Java
- Spring Boot
- JPA (Hibernate)
- Lombok
- Mysql

- Maven 3.6.0 o superior
